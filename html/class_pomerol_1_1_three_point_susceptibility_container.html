Detailed Description

Container for instances of ThreePointSusceptibility.

This class stores multiple \((i, j, \hat B)\)-elements of a 3-point susceptibility. A quadratic operator \(\hat B\) determines the channel the susceptibility is defined in, and is shared by all stored elements.
