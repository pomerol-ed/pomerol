Detailed Description

Expressions of quantum-mechanical operators.

The Pomerol::Operators namespace imports a few types and functions from libcommute. These include

  • The polynomial expression object, libcommute::expression.
  • The plus/minus Hermitian conjugate placeholder, libcommute::hc.
  • Factory functions for fermionic creation, annihilation and occupation operators ( \( \hat c^\dagger, \hat c, \hat n = \hat c^\dagger \hat c\)) with statically typed indices.
  • Factory functions for bosonic creation and annihilation operators ( \( \hat a^\dagger, \hat a\)) with statically typed indices.

There are also a few additional factory functions defined in this namespace.
